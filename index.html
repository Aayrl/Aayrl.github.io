Mill Map

A simple javascript page to keep track of spawn distributions and spawn timers for spawn points in the mobile game Pokemon Go.

The page data is populated from a Google Spreadsheet in real time using client-side javascript calls to the Google Spreadsheet API.

All graphs are powered by Google Charts API as well, with data populated from the Google Spreadsheet calls.
